# 测试清单

## 部署前准备

- [ ] 安装 Go 1.21+
- [ ] 从 @BotFather 获取 Bot Token
- [ ] 生成 32 字节加密密钥
- [ ] 复制并配置 config.env

## 基础功能测试

### 1. 启动测试
- [ ] 运行 `./start.sh` 或 `go run cmd/master/main.go`
- [ ] 查看是否显示 "✅ 主控Bot已启动"
- [ ] 查看是否显示 "🚀 开始接收消息..."

### 2. 首次设置管理员
- [ ] 在Telegram中向Bot发送 `/start`
- [ ] 检查是否收到欢迎消息
- [ ] 检查是否提示使用 `/setadmin` 设置管理员
- [ ] 发送 `/setadmin`
- [ ] 检查是否显示 "✅ 管理员设置成功！"

### 3. 管理员命令测试
- [ ] 发送 `/admin_help` 查看管理员帮助
- [ ] 发送 `/admin_stats` 查看系统统计

### 4. 套餐管理测试
- [ ] 发送 `/admin_add_plan`
- [ ] 输入套餐名称：测试套餐
- [ ] 输入价格：9.9
- [ ] 输入有效期：30
- [ ] 输入Bot数量：3
- [ ] 检查是否创建成功
- [ ] 发送 `/admin_list_plans` 查看套餐列表

### 5. 兑换码测试
- [ ] 发送 `/admin_generate_code`
- [ ] 输入套餐ID（从套餐列表获取）
- [ ] 输入生成数量：5
- [ ] 检查是否生成5个兑换码
- [ ] 发送 `/admin_code_list` 查看兑换码列表

### 6. 支付配置测试
- [ ] 发送 `/admin_payment_config`
- [ ] 输入API地址：https://test.com/api
- [ ] 输入商户ID：123456
- [ ] 输入商户密钥：test_key_123
- [ ] 检查是否保存成功
- [ ] 发送 `/admin_payment_status` 查看配置

### 7. AI配置测试
- [ ] 发送 `/admin_ai_config`
- [ ] 输入API地址：https://api.openai.com/v1/chat/completions
- [ ] 输入API密钥：sk-test123
- [ ] 输入模型名称：gpt-3.5-turbo
- [ ] 检查是否保存成功

### 8. 会话超时测试
- [ ] 发送 `/admin_add_plan`
- [ ] 等待6分钟（超过5分钟超时时间）
- [ ] 发送任意文本
- [ ] 检查是否提示会话已过期

### 9. 普通用户命令测试
- [ ] 发送 `/help` 查看帮助

### 10. 错误处理测试
- [ ] 发送不存在的命令 `/test123`
- [ ] 检查是否显示 "未知命令" 提示
